Credits & Attributions

Amble is built on the work of the open-data and open-source community.

This page credits the projects that make Amble possible. Their licences require attribution; we're glad to give it.

Map data

© OpenStreetMap contributors, available under the Open Database License.

OpenStreetMap is a free, editable map of the world maintained by hundreds of thousands of volunteers. Every walking path, park boundary, footbridge, and street lamp Amble routes through was mapped by someone, and we're grateful.

Map tiles

Map tile rendering by OpenFreeMap, a free, no-key map-tile service running an open style.

Map renderer

Native map rendering by MapLibre, an open-source fork of Mapbox GL.

Routing engines

Amble's primary route generation runs entirely on-device using a custom algorithm. For fallback routing, Amble uses one of the following:

  • BRouter — the default fallback. Open-source, community-hosted, no API key required.
  • OpenRouteService — alternative routing provider when BRouter is unavailable.
  • GraphHopper — alternative routing provider when configured.

Elevation data

Elevation profiles use OpenTopoData serving the SRTM 30m global digital elevation model.

Weather

Weather glances by Open-Meteo, free open-source weather forecasting.

Geocoding

Place name search by OpenStreetMap Nominatim.
